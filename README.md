# Sampai Akhir Bulan

Simulasi refleksi keputusan finansial. 10 skenario, 2 anonim, sekitar 7 menit.

## Coba

Buka [index.html](index.html) di browser, atau kunjungi:  
**https://nissafebrianis.github.io/sampai-akhir-bulan/**

## Tentang

Latihan singkat untuk melihat pola keputusan saat uang, tekanan sosial, dan emosi datang bareng. Bukan alat diagnosis.

- 10 skenario berbeda
- 2 persona (Anak SMA / Mahasiswa Awal)
- Pengeluaran otomatis untuk persona mahasiswa
- Penilaian risiko berdasarkan pola pilihan

## Jalankan Lokal

Buka `index.html` langsung di browser. Tidak perlu install apa pun.

## Deploy

Push ke branch `main`, GitHub Actions otomatis deploy ke GitHub Pages.
